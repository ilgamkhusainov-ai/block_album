# Block Puzzle MVP (RU)

## 1) Фиксация решений (по твоим ответам)
- Жанр: block puzzle + элементы casual strategy.
- Основная цель матча: набрать максимум очков.
- Конец матча: когда на поле нет валидных мест для фигур.
- Выдача фигур: 3 фигуры снизу.
- Обновление фигур: после хода добор до 3.
- Вариативность фигур: базовые формы + уникальные повороты и зеркальные варианты.
- Очистка: и линии, и зоны.
- Комбо: оба типа (мультиочистка за ход и серия успешных очисток подряд).
- Заряд силы: только от комбо.
- Бомбы: 3 вида (линейная горизонтальная, линейная вертикальная, область 3x3).
- Swap: меняет весь набор из 3 фигур.
- Подвижные блокеры: ходят каждые 2 хода игрока.
- Поведение блокеров: мешают, могут перекрывать, исчезать и появляться снова.
- Ввод новых правил: только когда это реально нужно для интереса.
- Кривая сложности: плавная.
- Подсказки: автоматом после паузы игрока.
- Обучение: без отдельного туториала, только подсказки.
- Визуальный стиль: яркий.
- Preview дропа: полупрозрачный «призрак» фигуры (как в Block Blast).
- Темп анимаций: медленный/сочный (читаемый).
- Сессия: 4-6 минут через фиксированную короткую цель + рост вариативности фигур.
- Текущая фиксированная цель уровня: 1000 очков.
- Meta между боями: временные задания + награды.
- Монетизация MVP: только rewarded ads.
- Главные приоритеты: ощущение управления и визуально-саундовая отдача.
- Целевое устройство для базовой верстки: iPhone 15 Pro Max.
- KPI первой недели: «хочется еще бой», средняя сессия > 4 минут, прохождение 10+ уровней без усталости.

## 2) Нерушимые правила MVP
- Одно новое правило за раз; не добавлять много механик одновременно.
- Любой элемент на экране должен помогать действию, а не отвлекать.
- Поражение должно восприниматься как «моя ошибка», а не «игра обманула».
- Кнопка рестарта/«еще бой» доступна сразу.
- Первые 60 секунд дают ощущение контроля и понятности.
- За простую постановку фигуры очки не начисляются.
- При очистке линии/зоны очки за отдельные клетки не начисляются (только линия/зона + прочие применимые источники).
- При очистке бустером: +1 за каждую уничтоженную клетку фигуры.
- Бустер не должен срабатывать по пустому тапу (только по занятой клетке или блокеру).
- Визуально показывать за что начислены очки на каждом ходе.

## 3) Core loop
1. Игрок получает 3 фигуры.
2. Перетаскивает фигуру на поле, видит ghost-preview позиции.
3. Фигура ставится, срабатывают очистки линий/зон.
4. Начисляется счет, считаются комбо, заряжается сила.
5. Каждый 2-й ход двигаются блокеры.
6. Добор фигур до 3.
7. Если валидных ходов нет -> конец матча -> результаты -> `PLAY AGAIN` или `SECOND CHANCE`; после победы используется `NEXT LEVEL`.

## 4) Минимальный состав механик (MVP)
- Поле 9x9.
- Базовые фигуры простой формы.
- Комбо-множитель.
- Блокеры (1 базовый тип поведения + спавн/деспавн).
- 3 бомбы + swap.
- Фиксированная цель по счету; усложнение по уровням идет через вариативность фигур.
- Рост вариативности: каждые 2 уровня до 24-го, с 25+ полный пул фигур.
- Перед стартом уровня: 5-секундный pop-up пула фигур; на Level 1 все фигуры `NEW`, далее подсветка `NEW/BOOST`.
- Автоподсказка после паузы (только после первого хода): проверяет все фигуры в руке, приоритет у очистки линий; если line clear нет, выбирает ход, который лучше приближает линии/зоны к заполнению; далее минимум остатков фигуры и максимум score/combo. Подсказка моргает 2 секунды и повторяется через 10 секунд бездействия.
- Rewarded ad после поражения: «второй шанс» или бонус.

## 5) Что не входит в MVP
- Сложная база/мета-прогрессия.
- PvP, соцфичи, кланы.
- Сложная экономика с несколькими валютами.
- События с большим объемом уникального контента.
